Créer de site web sur Android

Comment créer un site web sur Android ?

Dans cette article, je vais vous expliquer comment créer un site web en html et css sans avoir un ordinateur.

C’est quoi un ordinateur ?
Un ordinateur est un système de traitement de l'information programmable tel que défini par Alan Turing et qui fonctionne par la lecture séquentielle d'un ensemble d'instructions, organisées en programmes, qui lui font exécuter des opérations logiques et arithmétiques.

Qu'a besoin d'un ordinateur pour fonctionner ?
Un ordinateur a toujours besoin d’un système d’exploitation pour fonctionner,tel que : Windows, linux etc... C’est grâce à ces systèmes d’exploitations qu’on va pouvoir d’installer des logiciels pour fonctionner avec l’ordinateur.

Et c’est pareil pour un téléphone, on doit avoir aussi un système d’exploitation qui va vous permettre d’installer des applications plus intelligent pour travailler à notre besoin. Ce qui veut dire, ce qu’un ordinateur peut faire un smartphone Android peut le faire à 75% aussi.

Maintenant pour répondre à la question suivante, est-ce qu’on peut créer de site web à partie d’un smartphone Android ?

Qu’a-t-on besoin, pour créer un site web ? pour créer un site web, on a tout simplement besoin d’être un bon développer, un éditeur de texte et un navigateur, c’est tout. Alors sur un smartphone Android on peut avoir un éditeur de texte et un navigateur.

Maintenant, que disent les gens à propos de la création de site web sur Android, « on ne peut pas créer de site web sur Android, cause on ne peut pas gérer les media query ; ».
